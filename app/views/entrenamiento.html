Guía de Fundamentos Web

Alvaro Concha

Son los encargados de maquetar la estructura semántica del contenido (HTML), codificar el diseño en hojas de estilo (CSS) y agregar la interacción con el usuario (Javascript).

Frontend

En la época actual los Frontends tienen HTML5 y CSS3. Con HTML5, desde el frontend, es posible hacer geolocalización, dibujo vectorial, guardar datos en el disco del usuario, insertar audio y vídeo, entre otras cosas.

Con CSS3, se pueden crear diseños altamente complejos sin la necesidad de imágenes cortadas, sólo usando código. Bordes redondeados, sombras, degradados, fondos múltiples, entre otros. Por último, Javascript y sus frameworks añaden el componente de interactividad y conexión al servidor.

Es posible comunicarse con el backend y la base de datos sin recargar la página usando AJAX o WebSockets, para recibir esos datos y cambiar el diseño entero del sitio. jQuery hace todo esto fácil pero cabe aclarar que si bien es el más conocido, no es el único framework de Javascript.

Fundamentos de HTML5

HTML5

Giancarlo Sandoval

HTML5 es la actualización de HTML, el lenguaje en el que es creada la web. HTML5 también es un término de marketing para agrupar las nuevas tecnologías de desarrollo de aplicaciones web: HTML5, CSS3 y nuevas capacidades de Javascript que manejan diferentes APIs en el navegador.

La versión anterior y más usada de HTML, HTML4, carece de características necesarias para la creación de aplicaciones modernas basadas en un navegador. El uso fuerte de Javascript ha ayudado a mejorar esto, gracias a frameworks como jQuery UI, Sproutcore, entre otros.

Flash en especial ha sido usado en reemplazo de HTML para desarrollar web apps que superan las habilidades de un navegador: Audio, video, webcams, micrófonos, datos binarios, animaciones vectoriales, componentes de interfaz complejos, entre muchas otras cosas. Ahora HTML5 es capaz de hacer esto sin necesidad de plugins y con una gran compatibilidad entre navegadores modernos.

¿Cómo comenzar un documento HTML?

Lo primero que se declara siempre es el <!DOCTYPE>. Con esto definimos que versión de html usaremos.<!DOCTYPE html> define html5.Todo html sigue una estructura similar, primero se escribe la etiqueta <html></html> y dentro de el contiene una etiqueta <head></head> el cual define información del documento y otra <body></body> el cuál define información del cuerpo del documento:

1 <!DOCTYPE html> 2 <html lang="es"> 3 <head> 4 <meta charset="UTF-8"> 5 <title>Titulo</title> 6 </head> 7 <body> 8 <!-- Asi escribimos comentarios --> 9 </body> 10 </html>

Generalmente las etiquetas de abren <etiqueta> y cierran </etiqueta>, teniendo contenido por dentro, habiendo excepciones que no se cierran porque no tienen contenido, como <img>, <br> o <base>. Otro concepto fundamental es entender que cada etiqueta tiene atributos y cada atributo asume un valor:

<etiqueta atributo1=valor1 atributo2=valor2 ... atributon=valorN></etiqueta>

El atributo id por ejemplo define un identificador único en todo el documento que servira para asignarle estilos o scripts al componente de la etiqueta en que se encuentre gracias a los llamados selectores. Otro atributo que permite también acceder a elementos específicos es el atributo class. Para mayor detalle de los atributos se recomienda este enlace.

Dentro del head y del body también encontramos otras etiquetas que ayudan a definir la información del documento y de su cuerpo, respectivamente.

Giancarlo Sandoval

Fundamentos de CSS

CSS son las hojas de estilo en cascada, es decir, es el encargado de generar los estilos de nuestro documento HTML.

Si tú eres de las personas que dice que programa al momento de hacer un documento en CSS, te doy la triste y mala noticia de que esto es falso ya que tú nunca programas en CSS o en HTML, sino diseñas y modelas datos.

¿Cómo agregar CSS en HTML?

1. Directamente en HTML

Agregar CSS directamente en la etiqueta <style> del HTML

1 <style type="text/css"> 2 p { 3 color:green 4 } 5 </style>

2. En las etiquetas HTML

Agregar CSS en las etiquetas del HTML.

1 <p style="color:white;background:green;">Texto blanco fondo verde</p>

3. Link externo

Agregar un link a un archivo de CSS externo.

1 <link rel="stylesheet" type="text/css" href="estilos.css" />

4. Importando un archivo CSS

1 @import "estilos.css"

Estructura Básica

1 selector { 2 propiedad1:valor; 3 propiedad2:valor; 4 propiedad3:valor; 5 }

a. El selector identifica los elementos del HTML a los cuales se aplicará la regla.

b. Los pares propiedad-valor dentro de las llaves están separados por puntos y coma; y las propiedades están separadas de su respectivo valor por dos puntos.

c. Las propiedades definen lo que quieres modificar a los elementos seleccionados, por ejemplo, cambiar la posición, el color, los márgenes, el color de fondo, entre otros.

d. Los valores son los valores que quieres cambiar a cada propiedad de los elementos seleccionados. Los valores dependen de la propiedad, por ejemplo las propiedades que afectan al color pueden tomar nombres de color (en inglés) como red, gray, yellow, código del color en hexadecimal, como #009900 o valores RGB, como rgb(34,142,28). Las propiedades que afectan el ancho, la altura, la posición, el margen, etc pueden tomar valores medidos en pixel, ems, porcentaje, centímetros u otras unidades.

1 css 2 h1 { 3 color: red; 4 font-family: Arial; 5 font-size: 30px; 6 }

El selector h1 selecciona a todos los elementos <h1>en el documento HTML que usa este CSS para aplicar esta regla, a menos que existan selectores más específicos cuyas reglas sobrescribirán a esta.

Las propiedades en esta regla son el color del texto, la fuente del texto, y el tamaño de la fuente.

Los valores de estas propiedades son rojo, Arial, y 30 pixeles respectivamente.

Selectores

Hack Space Perú

Existen muchos tipos de selectores, los tres más usados que vas a encontrar son:

Selectores

1. Tag name: Selecciona los elementos del HTML de la respectiva etiqueta

1 p { 2 color: red; 3 }

2. Selector por clase: Selecciona los elementos del HTML que tienen un atributo class con el valor especificado.

1 HTML 2 <h1 class="ejemplo">Texto de ejemplo</p> 3 <div class="ejemplo">Contenido</div> 4 5 CSS 6 .ejemplo{ 7 color: blue; 8 }

3. Selector por id: Selecciona los elementos del HTML que tienen un atributo id con el valor especificado.

1 HTML 2 <h1 id="myid">Texto de ejemplo</p> 3 <div id="myid">Contenido</div> 4 5 CSS 6 #myid{ 7 color: blue; 8 }

Como sabemos que tú quieres ser muy bueno en CSS, la próxima semana incluiremos selectores avanzados, prepárate.

Agrupando Selectores

También es posible agrupar selectores que quieres que tengan el mismo estilo, por ejemplo en vez de escribir:

1 css 2 h1 { 3 color: blue; 4 font-size: 20px; 5 } 6 h2 { 7 color: blue; 8 font-size: 20px; 9 }

Donde los selectores h1 y h2 tienen el mismo estilo es posible agrupar los selectores separados por una coma:

1 css 2 h1,h2 { 3 color: blue; 4 font-size: 20px; 5 }

Selectores Universales

En CSS existe el selector universal * el cual selecciona todos los elementos de un tipo el cual no es muy usado ya que raras veces se desea que todos los elementos tengan el mismo estilo.

1 * { 2 margin: 0; 3 padding: 0; 4 }

Selectores descendentes

En algunas ocasiones se requiere dar estilos diferentes a elementos de un solo tipo pero que se diferencia por el elemento que los contiene.Un elemento A es descendiente de otro elemento B si el elemento A esta incluido en el elemento B.

HTML

1 <h1> 2 Lorem ipsum dolor sit amet, consectetur adipiscing elit. 3 <p>Praesent ac risus sed magna ornare pulvinar. </p> 4 </h1> 5 <p>Integer ex tortor, eleifend non porttitor eget, dictum eu elit.</p>

CSS

1 h1 p { 2 color: red; 3 }

En el ejemplo anterior existen dos etiquetas <p> pero se aplica solamente a la primera ya que esta está contenida dentro de la etiqueta h1 como se especifica en el selector del CSS.

Hay que resaltar que no necesariamente el elemento descendiente tiene que estar contenido inmediatamente dentro del elemento externo, es decir el elemento descendiente puede estar contenido dentro de otro elemento que a su vez este contenido en el elemento más externo.

1 <h1> 2 Lorem ipsum dolor sit amet, consectetur adipiscing elit. 3 <p>Praesent ac risus sed <strong>magna</strong> ornare pulvinar. </p> 4 </h1> 5 <strong>Integer ex tortor, eleifend non porttitor eget, dictum eu elit.</strong> 6 7 h1 strong { 8 color: red; 9 }

En el ejemplo anterior la etiqueta <strong>que deseamos aplicar el estilo está contenida dentro de una etiqueta

la cual a su vez está contenida en una etiqueta h1 por lo cual cumple la regla del selector del CSS.

En general la estructura para selecionar elementos descendientes es los selectores de cada elemento individual comenzando del mas externo hasta el más interno separadas por un espacio:

1 selector1 selector2 selector3 { 2 ... 3 }
